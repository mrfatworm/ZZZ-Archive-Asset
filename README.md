## Branch Rule

- dev: 測試版的主分支
- main: 上線正式版本
- f/*: 個人工作分支，例如更新青衣資料：f/qing_yi，此分支命名能看懂就行，最後後合併到 dev 後都會刪掉

main 與 dev 分支請發 PR (Pull Request) 指派對方 Review，勿直接推送與合併
